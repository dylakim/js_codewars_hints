Even or Odd

Instructions:

Create a function that takes an integer as an argument and returns "Even" for even numbers or "Odd" for odd numbers.



Tips

If a number is evenly divisible by 2, with no remainder, that means it is an even number

If a number is not evenly divisible by 2, and therefore has a remainder, that means it is an odd number

Use the remainder operator https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Remainder to evaluate if a value is even or odd.

The following examples, using the reminder operator, will evaluate to 0. If there is no remainder when the first number is divided by 2, this means the value is even.

  • 2 % 2
  • 4 % 2
  • 68 % 2
  • -36 % 2
  • 0 % 2

The following examples, using the remainder operator, will evaluate to something greater than 0. If there is some remainder when the first number is divided by 2, this means the value is odd.

  • 3 % 2
  • 7 % 2
  • 95 % 2
  • -49 % 2
